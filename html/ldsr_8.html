Description

Inverse Box-Cox transform

Usage

inv_boxcox(x, lambda)

Arguments

x

A numeric vector to be transformed
lambda

Lambda parameter

Value

The inverse Box-Cox

Examples

inv_boxcox(x = rnorm(10), lambda = 1)
