Weighted empirical distribution function

Description

Weighted empirical distribution function

Usage

.wecdf(x, w, type = c("dist", "surv"))

Arguments

x

vector of length n of values
w

vector of weights of length n
type

string, one of distribution function (dist) or survival function (surv)
