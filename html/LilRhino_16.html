Table_percent R Documentation

Table Percent

Description

Finds the acc of square tables.

Usage

Table_percent(in_table)

Arguments

in_table

a confusion matrix

Details

The table must be square

Note

make sure its square.

Author(s)

Travis Barton

Examples


true <- rep(1:10, 10)
test <- rep(1:10, 10)
test[c(2, 22, 33, 89)] = 1
Table_percent(table(true, test))
