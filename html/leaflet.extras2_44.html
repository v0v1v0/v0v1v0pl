goBackHistory R Documentation

goBackHistory

Description

If possible, will go to previous map extent. Pushes current extent to the "future" stack.

Usage

goBackHistory(map)

Arguments

map

a map widget object created from leafletProxy

Value

the new map object

References

https://github.com/cscott530/leaflet-history
