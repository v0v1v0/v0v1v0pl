linpkApp R Documentation

Runs the interactive shiny app.

Description

Runs the interactive shiny app.

Usage

linpkApp(...)

Arguments

...

Arguments passed to shiny::runApp().

Value

Called for its side effects.

Note

The app requires the following packages:

  • 'shiny'

  • 'shinyjs'

  • 'shinyAce'

  • 'dygraphs'

Make they are installed or the app won't work.

Examples

## Not run: 
linpkApp()

## End(Not run)
