mean_error R Documentation

Mean error

Description

Computes the average error between the input vectors.

Usage

mean_error(actual, predicted, na.rm = FALSE)

Arguments

actual

A numeric vector with the actual values.
predicted

A numeric vector with the predicted values. Each element in this vector must be a prediction for the corresponding element in actual.
na.rm

A logical value indicating whether NA values should be stripped before the computation proceeds. Defaults to FALSE.

Value

Returns a double scalar with the mean error value.

See Also

mean_error_pct(), mean_error_abs(), mean_error_abs_pct(), mean_error_sqr(), mean_error_sqr_root()

Examples

actual <- runif(10)
predicted <- runif(10)

mean_error(actual, predicted)
