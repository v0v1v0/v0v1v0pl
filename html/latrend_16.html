clusterNames R Documentation

Get the cluster names

Description

Get the cluster names

Usage

clusterNames(object, factor = FALSE)

Arguments

object

The lcModel object.
factor

Whether to return the cluster names as a factor.

Value

A character of the cluster names.

Examples
