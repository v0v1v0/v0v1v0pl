luz_callback_csv_logger R Documentation

CSV logger callback

Description

Logs metrics obtained during training a fiel on disk. The file will have 1 line for each epoch/validation.

Usage

luz_callback_csv_logger(path)

Arguments

path

path to a file on disk.
