yeast_top_table R Documentation

Top table for Yeast Lysate - UPS1 dataset

Description

Top table can be used for identifying the differential abundance analysis of proteins in the dataset.

Usage

yeast_top_table

Format

A data frame with 835 rows and 6 variables along with 'Fasta headers':

logFC

Log fold change values

AveExpr

Average intensity values

t

t-statistic values

P.Value

P-values

adj.P.Val

Adjusted P-values

B

B-statistic values
