get_log R Documentation

Get log object from a data item

Description

Get log object from a data item

Usage

get_log(data, logger = NULL)

Arguments

data

An R object.
logger

[character] scalar. Logger to return. Can be NULL when a single logger is attached.

Value

A logging object, or NULL if none exists.
