get_lake_names R Documentation

get lake names in GLTC dataset

Description

Lake names that are part of the Global Lake Temperature Collaboration can be accessed and returned. See associated publication and references therein for details including units and data provenance.

Usage

get_lake_names()

Value

a character vector of valid lake names from the Global Lake Temperature Collaboration dataset.

See Also

get_climate_names, get_metadata_names
