key R Documentation

key

Description

Creates a character key from arbitray R objects. For more details see vignette("listArray").

Usage

key(...)

Arguments

...

R objects

Value

a unique character key

Examples

key(1)
key(2,3)
key(1:3)
key(mean)
key('test')
key(letters[1:5])
key(list(1))
key(iris)
