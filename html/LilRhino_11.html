Percent R Documentation

Percent of confusion matrix

Description

For finding the accuracy of confusion matricies with true/pred values

Usage

Percent(true, test)

Arguments

true

The true values

test

the test values

Details

Make sure your strings have the right values and create a square matrix.

Value

the percent acc.

Author(s)

Travis Barton

Examples

true <- rep(1:10, 10)
test <- rep(1:10, 10)
test[c(2, 22, 33, 89)] = 1
Percent(true, test)
