find.na

Description

Finding missing values.

Usage

find.na( x )

Arguments

x

a numerical vector, matrix or data.frame.

Value

A numeric matrix with two columns.

Author(s)

Reza Mohammadi a.mohammadi@uva.nl and Kevin Burke kevin.burke@ul.ie

Examples

x = c( 2.3, NA, -1.4, 0, 3.45 )

find.na( x )
