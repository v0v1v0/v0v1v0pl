lsnstat: 'La Societe Nouvelle' API Access

Description

Tools facilitating access to the 'macro_data' service of the 'La Societe Nouvelle' API. It ensures an easy and fully-disclosed access to all macro-level data used in the 'La Societe Nouvelle' systems and the related metadata. Related API can be accessed from https://api.lasocietenouvelle.org/.
