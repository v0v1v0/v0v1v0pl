with_log_threshold R Documentation

Evaluate R expression with a temporarily updated log level threshold

Description

Evaluate R expression with a temporarily updated log level threshold

Usage

with_log_threshold(
  expression,
  threshold = ERROR,
  namespace = "global",
  index = 1
)

Arguments

expression

R command
threshold

log_levels
namespace

logger namespace
index

index of the logger within the namespace

Examples

## Not run: 
log_threshold(TRACE)
log_trace('Logging everything!')
x <- with_log_threshold({
  log_info('Now we are temporarily suppressing eg INFO messages')
  log_warn('WARN')
  log_debug('Debug messages are suppressed as well')
  log_error('ERROR')
  invisible(42)
}, threshold = WARN)
x
log_trace('DONE')
