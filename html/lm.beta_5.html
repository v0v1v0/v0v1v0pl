summary.lm.beta R Documentation

Summarize Standardized Linear-Model-Object

Description

S3-method summary for object lm.beta.

Usage

## S3 method for class 'lm.beta'
summary(object, standardized = TRUE, ...)

Arguments

object

An R object of class lm.beta
standardized

Logical. Should the standardized values be integrated?
...

Additional arguments to pass to summary.lm

Details

If standardized=FALSE, the standard summary.lm-method is called, else (the standard value) the standardized regression coefficients are added into the coefficient table.

The additional arguments are passed to summary.lm.

Value

Adapted summary.lm-object, in case of standardized=TRUE with additional class summary.lm.beta.

Author(s)

Stefan Behrendt, r@behrendt-stefan.de

See Also

lm.beta for creating the lm.beta-object.

Examples

## Taken from lm help
##
## Annette Dobson (1990) "An Introduction to Generalized Linear Models".
## Page 9: Plant Weight Data.
ctl <- c(4.17,5.58,5.18,6.11,4.50,4.61,5.17,4.53,5.33,5.14)
trt <- c(4.81,4.17,4.41,3.59,5.87,3.83,6.03,4.89,4.32,4.69)
group <- gl(2, 10, 20, labels = c("Ctl","Trt"))
weight <- c(ctl, trt)
lm.D9 <- lm(weight ~ group)

# standardize
lm.D9.beta <- lm.beta(lm.D9)
summary(lm.D9.beta)
summary(lm.D9.beta,standardized=FALSE)
