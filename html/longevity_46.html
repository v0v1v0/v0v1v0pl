hweibull R Documentation

Hazard function of the Weibull distribution

Description

Hazard function of the Weibull distribution

Usage

hweibull(x, shape, scale = 1, log = FALSE)

Arguments

x

vector of quantiles
shape

shape parameter
scale

scale parameter, default to 1
log

logical; if TRUE, returns the log hazard

Value

a vector of (log)-hazard
