Ekc R Documentation

CO2 emissions

Description

CO2 emissions per capita and GDP per capita in 2005

Usage

data(Ekc)

Format

A data frame with 183 rows and 4 variables:

ccode

country code

cname

country name

gdppcppp

GDP per capital, ppp adjusted (USD)

co2pc

CO2 emissions per capita (ton)

Author(s)

NA

Source

http://wdi.worldbank.org
