glmerLaplaceHandle R Documentation

Handle for glmerLaplace

Usage

Arguments

pp

resp

lmResp object
nAGQ

see glmer
tol

tolerance
maxit

maximum number of pwrss iterations
verbose

display optimizer progress

Value
