print.larf R Documentation

Print Results of the Estimated LARF

Description

Methods to display brief results of a larf object.

Usage

## S3 method for class 'larf'
print(x, digits = 4, ...)

Arguments

x

an object of class "larf" as fitted by larf.
digits

The number of significant digits to be printed in the reports of the results.
...

currently not used.
