sw_lucene R Documentation

Lucene Stopword List

Description

A dataset containing a character vector of Lucene's stopwords used in StopAnalyzer.ENGLISH_STOP_WORDS_SE.

Usage

data(sw_lucene)

Format

A character vector with 33 elements
