plot_dl_test R Documentation

Plots the Dominguez-Lobato test.

Description

Plots the Dominguez-Lobato test.

Usage

plot_dl_test(x)

Arguments

x

An object of class 'dl_test'.

Value

Plot of type ggplot.

Examples

x <- 1:10
y <- 1:10
model <- lm(y~x-1)
dl_test <- dominguez_lobato_test(model)
plot_dl_test(dl_test)
