sim18cfa0 R Documentation

Simulated CFA data with LI

Description

CFA data simulated based on 18 items, 3 factors and local independence; factorial correlation \Phi=.3.

Usage

sim18cfa0

Format

A list with components:

dat

A dataset with simulated responses of 1000 individuals to 18 items

qlam

Loading pattern and values used to simulated the data
