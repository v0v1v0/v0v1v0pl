T2prob R Documentation

Convert a Vector of T-year Return Periods to Annual Nonexceedance Probabilities

Description

This function converts a vector of T-year return periods to annual nonexceedance probabilities F

F = 1 - \frac{1}{T}\mbox{,}

where 0 \le F \le 1.

Usage

T2prob(T)

Arguments

T

A vector of T-year return periods.

Value

A vector of annual nonexceedance probabilities.

Author(s)

W.H. Asquith

See Also

prob2T, nonexceeds, add.lmomco.axis

Examples

T <- c(1, 2, 5, 10, 25, 50, 100, 250, 500)
F <- T2prob(T)
