Class Lagged1d

Description

Class Lagged1d.

Objects from the Class

Objects can be created by calls of the form Lagged(v) or new("Lagged1d", data = v), where v is a vector. new("Lagged1d", ...) also works.

Slots

data:

Object of class "vector".

Extends

Class "Lagged", directly.
