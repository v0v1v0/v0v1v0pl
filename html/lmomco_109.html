grv2prob R Documentation

Convert a Vector of Gumbel Reduced Variates to Annual Nonexceedance Probabilities

Description

This function converts a vector of Gumbel reduced variates (grv) to annual nonexceedance probabilities F

F = \exp(-\exp(-grv))\mbox{,}

where 0 \le F \le 1.

Usage

grv2prob(grv)

Arguments

grv

A vector of Gumbel reduced variates.

Value

A vector of annual nonexceedance probabilities.

Author(s)

W.H. Asquith

See Also

prob2grv, prob2T

Examples

T <- c(1, 2, 5, 10, 25, 50, 100, 250, 500); grv <- prob2grv(T2prob(T))
F <- grv2prob(grv)
