common_env R Documentation

Find common environment in list of lazy objects.

Description

If no common environment is found, will return baseenv().

Usage

common_env(dots)

Arguments

dots

A list of lazy objects

Examples

common_env(lazy_dots(a, b, c))

f <- function(x) ~x
common_env(list(f(1)))
common_env(list(f(1), f(2)))
