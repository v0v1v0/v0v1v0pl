mat.to.laplacian R Documentation

Computes Laplacian matrix.

Description

Computes Laplacian matrix.

Usage

mat.to.laplacian(M1,type)

Arguments

M1

p \times p matrix
type

Laplacian types: 1) "normalized" (default) - normalized Laplacian, 2) "combinatorial" - combinatorial Laplacian

Value

L

Laplacian
