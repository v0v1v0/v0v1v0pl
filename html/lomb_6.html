ggamma R Documentation

Utility function called by pbaluev()

Description

From astropy.timeseries

Usage

ggamma(N)

Arguments

N

A positive number

Value

sqrt(2 / N) * exp(lgamma(N / 2) - lgamma((N - 1) / 2))

Author(s)

Thomas Ruf thomas.p.ruf@me.com
