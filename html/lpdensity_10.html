Ggenerate R Documentation

Internal function.

Description

Generate matrix.

Usage

Ggenerate(p, low = -1, up = 1, kernel = "triangular")

Arguments

p

Nonnegative integer, polynomial order.
low, up

Scalar, between -1 and 1, the region of integration.
kernel

String, the kernel function.

Value

A (p+1)-by-(p+1) matrix.
