Generic Contrast Test Functions

Description

Generic functions for tests contrasts.

Usage

contest(model, L, ...)

contest1D(model, L, ...)

contestMD(model, L, ...)

Arguments

model

a model object.
L

a contrast vector or matrix.
...

additional arguments passed to methods.
