Correctness linters

Description

Linters highlighting possible programming mistakes, such as unused variables.

Linters

The following linters are tagged with 'correctness':

  • duplicate_argument_linter

  • equals_na_linter

  • missing_argument_linter

  • namespace_linter

  • object_usage_linter

  • package_hooks_linter

  • sprintf_linter
