lero.lero R Documentation

Generate 'Lero Lero' quotes

Description

Generates a quote from 'Lero Lero' database (see <http://www.lerolero.com/> for more information).

Usage

lero.lero(modo = "original")

Arguments

modo

Indicates a mode. One of "original", "psicanalitico", or "programador".

Examples

# lero.lero()
# lero.lero(modo = "programador")
# lero.lero(modo = "psicanalitico")
