.logrank_trafo2 R Documentation

Logrank transformation function for LTRC data

Description

.logrank_trafo transforms Surv(time1, time2, event) objects into logrank scores, which will be used later in the tree algorithm. It is not designed to be used by users, not for internal used of LTRCIT function.

Usage

.logrank_trafo2(x2)

Arguments

x2

A vector Surv (Surv(time1, time2, event)) objects

Value

Logrank scores of LTRC objects
