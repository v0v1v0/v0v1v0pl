freq_last_names R Documentation

Frequent U.S. Last Names

Description

A dataset containing frequent last names based on the 1990 U.S. census.

Usage

data(freq_last_names)

Format

A data frame with 14,840 rows and 2 variables

Details

  • Surname. A last name

  • prop. The proportion
