to_date R Documentation

Convert event endpoints to dates

Description

Convert event endpoints to dates

Usage

to_date(e, origin = as.Date("1970-01-01"))

Arguments

e

Event table or atomic vector.
origin

Date object (see as.Date).

Examples

t <- as.Date("1970-01-01") + 0:4
e <- events(t)
to_date(e)
to_date(e$from)
