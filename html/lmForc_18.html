h_ahead R Documentation

Get the h_ahead slot of a h_ahead object

Description

h_ahead takes a Forecast object and gets the h_ahead vector of the forecast.

Usage

h_ahead(Forecast)

Arguments

Forecast

Forecast object.

Value

Vector of h_ahead values stored in the Forecast object.
