plot_reset_test R Documentation

Plot the reset test.

Description

Plot the reset test.

Usage

plot_reset_test(x)

Arguments

x

An object of class 'reset_test'.

Value

Plot of type ggplot.

Examples

x <- 1:10 + rnorm(10)
y <- 1:10
model <- lm(y~x-1)
r_test <- reset_test(model)
plot_reset_test(r_test)
